# CondenseTech.Ofd
SDK to handle OFD (Chinese version of PDF) documents, which mainly used by tax authorities generating electronic invoices.
Can be used with following framework:
+ .net4.8
+ .net5.0
+ .net6.0
+ .net7.0
+ .net standard 2.1
## Features
+ Load OFD documents
+ Verify digital signature
+ Render most type of elements

## Roadmap
+ Support rendering more types of elements
+ Extraction of electronic invoice key fields
